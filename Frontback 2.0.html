Windows

Чтобы включить режим максимальной производительности, запустите командную строку или консоль Windows PowerShell от имени администратора и выполните следующую команду:
powercfg -duplicatescheme e9a42b02-d5df-448d-aa00-03f14749eb61

Чтобы отключить режим максимальной производительности, запустите командную строку или консоль Windows PowerShell от имени администратора и выполните следующую команду:
powercfg -restoredefaultschemes

Список служб для отключения:
• Биометрическая служба Windows - Отключаем если на компьютере нет сканера отпечатка пальцев или любого другого биометрического сканера.
• Диспетчер проверки подлинности Xbox Live - Отключаем если не пользуемся Xbox Live.
• Контейнер службы Microsoft Passport - Отключаем если не пользуемся виртуальными смарт-картами TPM.
• Политика удаления смарт-карт - Отключаем если не пользуемся смарт картами.
• Сетевая служба Xbox Live - Отключаем если не пользуемся Xbox Live.
• Служба перечисления устройств чтения смарт-карт - отключать если вы не пользуетесь смарт картами и если вы не используете интерфейс WinRT.
• Служба шифрования дисков BitLocker - Отключать если не используете шифрование Битлокер, отключать тем у кого проблемы с вирусами шифровальшиками.
• Смарт-карта - Отключать если не пользуетесь смарт картами.
• Сохранение игр на Xbox Live - Отключаем если не пользуемся Xbox Live.

В Windows 10 по умолчанию присутствует поддержка кодеков H.264 в контейнере MKV, аудио без потерь в формате FLAC, а также видео, закодированного с использованием кодека HEVC/H.265 (который, судя по всему, в ближайшем будущем будет использоваться для большинства 4K видео).

И если тебя этот элемент не устраивает, просто прекращаешь работу функции через ретерн. Как вариант if(!event.target.classList.contains('game-item')) return
